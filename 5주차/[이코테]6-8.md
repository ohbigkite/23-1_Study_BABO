### 문제풀이 핵심아이디어 또는 새롭게 알게된 내용: 
    리스트 변수가 하나 있을 때 내부 원소를 바로 정렬 -> 리스트 객체의 내장함수 sort() 이용
    
### 코드의 시간복잡도와 그 이유:
  O(NlogN) : sort 함수는 입력의 크기가 커질 경우 효율적으로 정렬하기 위해 퀵 정렬과 같은 진화된 정렬 방식을 사용한다고 하네요~ 그래서 NlogN의 시간 복잡도를 가집니다!
    
    
### 주석이 가득 담긴 코드:
```python
array = [7, 5, 9, 0, 3, 1, 6, 2, 4, 8]

array.sort()
print(array)
# 밑에 지우면 안올라가요!
```
